# node_desafios

## Instalação
- Após clonar o repositório, instale as dependências com:

```bash
npm install
```

## Executar todos os testes
- Para rodar toda a suíte de testes utilizamos o Jest:

```bash
npm test
```

## Executar um teste específico
- Use o Jest diretamente para especificar o arquivo desejado:

```bash
npx jest tests/scriptEstoque.test.js
npx jest tests/scritptMaxNumber.test.js
```
